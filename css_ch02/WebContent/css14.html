  • xxSTARTxx

class 속성[~로 끝난다]요소를 선택

  • ENDxx

  • xxENDxx

class 속성[~를 포함한다]요소를 선택

  • CNTxx
  • xxCNT
  • xxCNTxx

  • 속성연습1
